# ⚡ Pokémon Chess

Chess with Pokémon as the pieces — play in any browser, on PC or mobile.

**Team Mewtwo (White):** Mewtwo · Charizard · Snorlax · Alakazam · Rapidash · Pikachu
**Team Mew (Black):** Mew · Gengar · Onix · Hypno · Arcanine · Eevee

## Game modes

- 🤖 **vs Computer** — play against a built-in AI
- 👥 **2 Players** — pass-and-play on one device
- 🌐 **Online** — create a game, share the code or link, and a friend can join from any phone or PC (peer-to-peer via WebRTC, no account needed)

Pawns that reach the far side *evolve* into the team's Queen. Captured Pokémon faint to the sidebar.
